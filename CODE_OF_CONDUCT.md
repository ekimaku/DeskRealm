# Code of conduct

Keep discussions respectful, specific, and useful.

## Expected behavior

- Be kind and direct.
- Critique code and behavior, not people.
- Provide logs, steps, and context when reporting issues.
- Respect that DeskRealm touches sensitive Windows Shell behavior and must remain conservative.

## Unacceptable behavior

- Harassment or insults.
- Publishing private information.
- Pressure to add unsafe behavior without documentation or safeguards.
- Dismissing bug reports without investigation.

## Enforcement

Maintainers may close, hide, or remove discussions that make the project unsafe or hostile.
