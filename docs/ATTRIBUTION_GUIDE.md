# Attribution guide

DeskRealm is licensed under Apache-2.0.

## If you redistribute DeskRealm or a derivative work

Preserve:

- `LICENSE`
- `NOTICE`
- copyright/attribution notices in source files and documentation
- clear notices on modified files where applicable

## If DeskRealm inspires your project but you do not copy code

The Apache-2.0 license cannot impose attribution for pure inspiration that does not copy protected material. However, the author kindly requests credit.

Suggested credit:

```text
Inspired by DeskRealm by Ayahua:
https://github.com/ekimaku/DeskRealm
```

## Citation

GitHub can display a “Cite this repository” action when a `CITATION.cff` file exists at the repository root. Use the provided `CITATION.cff` and update the repository URL before publishing.

## Suggested README credit for derivative projects

```markdown
This project uses or derives from DeskRealm by Ayahua, licensed under Apache-2.0.
Original project: https://github.com/ekimaku/DeskRealm
```
