# Installation

DeskRealm is a Windows tray utility. For normal users, prefer the release artifacts generated by GitHub Actions instead of building from source.

## Recommended: portable release ZIP

1. Open the latest GitHub Release.
2. Download `DeskRealm-<version>-win-x64-portable.zip`.
3. Extract it somewhere stable, for example:

   ```text
   %LOCALAPPDATA%\Programs\DeskRealm
   ```

4. Run `DeskRealm.App.exe`.
5. In the tray menu, enable **Démarrer avec Windows** if you want it to start automatically.

## Install bundle

Release builds also provide:

```text
DeskRealm-<version>-win-x64-install-bundle.zip
```

This bundle contains the published app plus helper scripts:

```powershell
powershell -ExecutionPolicy Bypass -File .\Install-DeskRealm.ps1
```

Optional flags:

```powershell
powershell -ExecutionPolicy Bypass -File .\Install-DeskRealm.ps1 -StartAfterInstall -StartWithWindows
```

Uninstall:

```powershell
powershell -ExecutionPolicy Bypass -File .\Uninstall-DeskRealm.ps1
```

By default, uninstall preserves `%APPDATA%\DeskRealm` and `%LOCALAPPDATA%\DeskRealm` so your config, icon layouts and logs are not destroyed. Use `-RemoveUserConfig` only if you intentionally want a full cleanup.

## Build from source

Install the .NET 8 SDK, then run:

```powershell
.\scripts\Build-Release.ps1
```

The local build output is:

```text
.\dist\DeskRealm\DeskRealm.App.exe
```

## Safety checklist

Before first run:

- Keep `rejectOneDriveDesktop` enabled unless you know exactly what you are doing.
- Back up important Desktop files.
- Keep `scripts/Restore-Desktop.ps1` available.
- Disable Windows Desktop icon auto-arrange if you want icon positions to remain stable.
