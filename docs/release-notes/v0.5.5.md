# DeskRealm v0.5.5

## Verified chunked icon placement

This hotfix improves desktop icon restoration reliability when some icons remain at their previous position after a virtual desktop switch.

### Changed

- Restores icons in small Shell batches instead of one large batch.
- Verifies the actual position of restored icons after placement.
- Retries only unresolved icons up to three times.
- Logs any icon that still refuses to reach its target position after verification.

### Why

On some Explorer/Desktop states, `IFolderView::SelectAndPositionItems` can return success while one or two visible icons are not yet at their requested location. DeskRealm now treats icon placement as a verified operation instead of a fire-and-forget operation.
