# DeskRealm v0.5.2

DeskRealm v0.5.2 is a focused icon-layout hotfix.

## Fixed

- Prevents cross-desktop icon position contamination when identical icons exist on multiple realms.
- Avoids saving a layout if the current Windows virtual desktop does not match the realm currently assigned as the known Desktop folder.
- Keeps safe pre-switch saves for DeskRealm hotkey navigation.

## Notes

For best results after upgrading, manually save the icon layout once on each realm where the same icon exists but should keep a different position.
