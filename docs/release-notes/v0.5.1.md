# DeskRealm v0.5.1

Quiet icon layout hotfix.

## Highlights

- Disabled background icon layout autosave by default.
- Added config migration to disable existing v0.5.0 periodic autosave settings automatically.
- Kept automatic layout persistence on desktop switch and before exit restore.
- Removed post-restore Shell refresh after icon placement to avoid Explorer reordering icons after DeskRealm restores them.
- Updated documentation to describe the quiet save model.

## Why

The v0.5.0 periodic icon layout worker could briefly trigger the Windows busy cursor every autosave interval. This made DeskRealm visible in day-to-day use instead of disappearing into the background. v0.5.1 prioritizes flow: no periodic icon polling during normal idle use.
