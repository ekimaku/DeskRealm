# DeskRealm v0.5.7

First-run Desktop import wizard.

## Added

- First-run wizard for new installations.
- Choose which Windows virtual desktop receives the current Windows Desktop.
- Optional move of current Desktop files and shortcuts into the selected realm.
- Optional save of current icon positions as the selected realm's initial layout.
- Strict conflict checks before moving files: no silent overwrite and no hidden merge.
- Config version `5` for first-run import settings.

## Upgrade behavior

Existing DeskRealm installations are automatically marked as import-completed during config migration, so the wizard does not suddenly appear after updating from an earlier version.

## Icon layout engine

This release keeps the validated v0.5.6 icon layout engine: display topology variants, DPI/scale awareness, delayed/verified restore, and Shell identity fallback for repeated shortcuts.
